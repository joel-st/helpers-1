Simple grid layouts using colspan

The code in this example was derived from clearleft.co.uk/styleguide and uses class names to replicate the HTML table layouts, including colspan.

Usage

  1. Call @import '/Library/WebServer/webroot/frappant_helpers/less/colgroup'; in your LESS file.
  2. Use the class names colgroup-2, colgroup-3 or colgroup-4 on the row.
  3. Use class name col with optional additional name colspan-2 or colspan-2 on the “column”.

But for some reason, you and I react the exact same way to water. We swallow it too fast, we choke. We get some in our lungs, we drown. However unreal it may seem, we are connected, you and I. We're on the same curve, just on opposite ends.

Bourbon Neat grid system

Bourbon is pre-installed within Codekit, so you only need to call @import "bourbon", "neat"; in your Sass file. This makes the grid mixins available for your own code.

Bourbon's documentation is here and its own grid layout examples are here.

The SCSS file in this folder is commented to explain how it is implemented.

KMU responsive layout starting point

This is a starting point responsive template package for the most common layout used by small and medium businesses. A more extensive version of this template is in use at biketowork.ch.

This layout is not intended for use as-is. It is a starting point from which a final template can be created.

Layout variations

  • Mobile (<768px): single flexible column
  • Tablet (768px-980px): two flexible columns
  • Desktop (980px>): three fixed columns

Break points

The most common breakpoints are in use in this template, but can of course be modified. The standards are 640px, 768px and 980px. A 1024px break point is not in use, as the maximum width of this basis layout is 1024px. Again, this can be extended as required.

Typography

Typography is based on a 13px base font size at mobile resolution, a 16px base font size at tablet resolution and a 12px base font size at desktop resolution. These settings are in order to maintain good legibility and, as near as possible, common line-ending breaks in the various resolutions.

Legibility has been tested on all devices using the standard Helvetica Neue font, with a fallback to Helvetica and Arial. Adjustments may be required for other font faces or other font weights.

Menu

The menu is hidden by default in mobile view (<768px). It can be toggled into view by means of a button which is visible at <768px.
